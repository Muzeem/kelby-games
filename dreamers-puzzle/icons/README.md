# App Icons Required

For Microsoft Store submission, you need these icon sizes:

## Required Icons
- icon-16.png (16x16)
- icon-32.png (32x32)
- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

## Design Guidelines
- Use a simple, recognizable design
- Works well at small sizes
- Suggested: Puzzle piece with golden accent
- Background: Dark (#1a1a2e) or transparent
- Accent color: Gold (#c9a227)

## Tools to Create Icons
- Figma (free)
- Canva
- Adobe Illustrator
- GIMP (free)

## Quick Method
1. Create a 512x512 master icon
2. Use an icon generator like:
   - realfavicongenerator.net
   - favicon.io
   - pwa-asset-generator (npm package)
